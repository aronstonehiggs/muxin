   

# 入埃及记

开罗小街窄巷

五色斑斓杂货铺

我不买什么

喜欢看，看看

与金字塔相反的

零零碎碎日用品

埃及像爿露天的店

尼罗河，长长的市河

斯芬克斯大掌柜哪

金字塔不是日用品

多么令人幸灾乐祸哟

亚述波斯希腊罗马

阿拉伯土耳其，这些劫掠者

两手空空踅进了历史

在希腊之前，你们的

七圣音、竖琴，迷惑过我

你们的宅，无窗无光

你是柔土研制的陶人

将你的脸拨侧在枕上

你的双肩平贴在毡毯上

按捺成壁画的正面律

瞳彩金褐，呼吸剧促

含族抑阿拉伯的苗裔哟

以我一身烈火加诸你柔土

金字塔太重，强盗搬不动

永恒太贵，谁也买不起

我独揽你粗犷中的秀媚
